异步页面渲染成静态页面


2017-08-22


返回目录

原有项目使用了artTemplate模板引擎，引擎虽好，但是对SEO来说实在不佳。按理说模板引擎也没什么，但最要命的是数据都是ajax拉回来的，因此又多了一关。

最佳的思路就是整个页面拉回来，让js都跑一遍然后得出最后的结果页面。但是不是dom环境下，window/dom等等的object都缺少了。
